📊 Connect Your Google Sheet

  1. Open Google Sheets and create a new spreadsheet.
  2. Row 1 must be exact headers (copy-paste this exactly):
    name price category image available customizable shortDesc fullDesc
  3. Fill in your products from row 2 onwards. See column guide below.
  4. Publish the sheet: File → Share → Publish to web → Sheet1 → CSV → Publish → Copy the URL.
  5. Paste the URL in the box below and click Save.
ColumnWhat to writeExample
nameProduct nameGeode Resin Tray
priceNumber only (₹)1299
categoryOne word/phraseTray
imagePaste a public image URLhttps://i.imgur.com/abc.jpg

customizableTRUE or FALSETRUE

fullDescFull description for popupMade with premium resin…

💡 To add/edit/delete products: open your Google Sheet, make changes, save — then click Reload Products (or visitors will see updates on their next page load).

📷 How to add product images

Google Sheets only stores text, so images must be hosted online. Here are free options:

  1. Imgur (easiest): Go to imgur.com/upload → upload photo → right-click image → Copy image address → paste in Sheet.
  2. Google Drive: Upload photo → right-click → Share → Anyone with link → copy link. Then convert to direct URL: replace drive.google.com/file/d/FILE_ID/view with drive.google.com/uc?id=FILE_ID
  3. Cloudinary (best quality): Free account at cloudinary.com → upload → copy URL.

🔑 Change Admin Password

Password is saved in your browser's localStorage. You'll need to update it on each device you use.
